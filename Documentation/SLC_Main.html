LANDFORMS 01 terrain variables

This tool generates the four key variables required to perform the surface elements classification: slope, ruggedness (vrm), finescale and broadscale bathymetric position index (BPI).

Slope calculates the gradient change of the surface in degrees. Slope is calculated using the QGIS native 'Slope' tool.

Ruggedness (vrm) calculates the variation of a surface as a representation of roughness, with output values ranging from 0 (no variation) to 1 (complete variation). Ruggedness is calculated using the SAGA 'Vector Ruggedness Measure (VRM)' tool and uses a 'square' search mode with radius 1.

Bathymetric Position Index (BPI) calculates relative elevation across a seascape, where large positive values represent higher elevation areas of the dataset, large negative values represent area of lower elevation, and values around zero represent flatter areas. BPI is calculated using the methodology from the 'Slope Position' tool in the Geomorphometry and Gradient Metrics v2.0 toolbox (Evans et al. 2014) and is calculated at user-defined broad and fine scales. Broadscale BPI uses a higher value for the window size, while finescale BPI uses a lower value for the window size.

Default values for the BPI window sizes are based on a 5 m DEM smoothed with x3 iterations and are intended as a guide.

This tool is intended for users to explore the terrain variables and determine suitable threshold settings.

Evans, J., Oakleaf, J., Cushman, S., 2014. An ArcGIS Toolbox for Surface Gradient and Geomorphometric Modeling, Version 2.0-0. Available online: https://github.com/jeffreyevans/GradientMetrics
