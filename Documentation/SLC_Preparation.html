PREP clip DEM

Clip DEM raster to specified depths.

Input parameters

DEM

Input DEM

DEM Label [Optional]

Label for output directory i.e. PREP_[DEM Label].

If left blank clipped DEM raster dataset is output to origin directory.

Shallow cutoff value (m)

Shallowest depth to clip to as a positive number.

Deepest cutoff value (m) [Optional]

Deepest depth to clip to as a positive number. If left blank maximum depth in existing DEM is used.

Outputs

A clipped GeoTIFF named DEM_[ShallowDepth]m_[DeepDepth]m (i.e. DEM_20m_59m.tif for a DEM clipped below 20m with a maximum depth of 59m) is output.


Algorithm version: V1.0

Last Modified: 15-Oct-2025


PREP make DEM from xyz

Converts xyz data to GeoTIFF raster.

Input parameters

Depth (xyz)

Input file with depths as xyz.

DEM Label

Label for output DEM and location i.e. test.

Convert Depths

Check box to convert postive depths in xyz file to negative (default is yes). i.e. xyz output from HYPACK.

Outputs

A GeoTIFF named DEM_[DEM Label] is written to a directory called PREP_[DEM Label] in the project directory.


Algorithm version: V1.1

Last Modified: 19-Nov-2025


PREP smooth DEM

Iteratively smooth DEM raster dataset.

NOTE: This tool uses the WhiteboxTools - MedianFilter

Input parameters

DEM

Input raster dataset (DEM) as a GeoTIFF

DEM Label

Label for output directory i.e. PREP_[DEM Label].

If left blank smoothed DEM raster dataset is output to origin directory.

Number of Iterations

Number of iterations to smooth over (i.e. 3).

Outputs

A smoothed GeoTIFF named DEM_s[Number of iterations] (i.e. DEM_s3.tif for 3 iterations) is output.


Algorithm version: V1.1

Last Modified: 19-Nov-2025
